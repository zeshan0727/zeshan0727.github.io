# PhoneAura 0.4.6

Stable device-tested Concept D redesign for iOS 16.

Published variants:

- RootHide: `iphoneos-arm64e`
- Rootless (Dopamine-style environments): `iphoneos-arm64`

Both packages use the same identifier and version so Sileo selects the correct build for the active jailbreak architecture. The release includes the full-stage Favorites, Recents, Contacts and Keypad design, inline contact details, in-app favorite selection, Siri/My Info My Card lookup, PhoneAura Studio and PreferenceLoader settings.
